Frequently Asked Questions

How do I enter which pages to extract?

Type page numbers separated by commas, and use a hyphen for a range. For example, "1, 3, 5-7" extracts page 1, page 3, and pages 5 through 7.

What order will the extracted pages be in?

They'll appear in the order you type them, so you can also use this tool to reorder select pages — for example, "5, 1, 3".

Can I extract the same page more than once?

Yes, if you list a page number more than once, it will appear multiple times in the output.

What if I want to delete pages instead of keeping them?

Use the Remove Pages tool instead — it lets you specify which pages to delete rather than which to keep.
