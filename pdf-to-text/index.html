Frequently Asked Questions

Why is my extracted text file empty?

This happens when the PDF is scanned or made up of images rather than real text — for example, a photo of a document saved as a PDF. This tool can only extract text that's actually stored as text in the file, not text within an image.

Does this preserve formatting like bold or tables?

No. The output is plain text only — formatting, fonts, and layout are not preserved, only the words themselves.

Can I extract text from scanned documents?

Not with this tool. Scanned PDFs need OCR (optical character recognition) to detect text within images, which this tool does not currently perform.

Is my file uploaded anywhere?

No. Text extraction happens entirely in your browser — nothing is uploaded to a server.
